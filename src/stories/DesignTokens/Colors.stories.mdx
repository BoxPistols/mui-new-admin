# カラーシステム

このプロジェクトのカラーシステムは、統一されたユーザーエクスペリエンスと視覚的な一貫性を提供するために設計されています。

## 設計思想

### ブランドアイデンティティ
- **プライマリブルー**: 信頼性とプロフェッショナルさを表現
- **アクセシビリティ重視**: WCAG 2.1 AA以上のコントラスト比を確保
- **ダークモード対応**: ライト・ダークテーマ両方で最適化

## 使用方法

### 開発者向けガイドライン

1. **プライマリアクション**: Brand 400を使用
2. **セカンダリアクション**: Gray 600を使用
3. **テキスト**: 
   - プライマリテキスト: Gray 800 (Light) / White (Dark)
   - セカンダリテキスト: Gray 600 (Light) / Gray 400 (Dark)
4. **背景**: 
   - メイン背景: hsl(0, 0%, 99%) (Light) / Gray 800 (Dark)
   - カード背景: hsl(220, 35%, 97%) (Light) / hsl(220, 30%, 20%) (Dark)

### アクセシビリティ

- すべてのカラーコンビネーションはWCAG 2.1 AA基準を満たしています
- テキストと背景のコントラスト比は4.5:1以上を確保
- カラーブラインドネスへの配慮として、色のみに依存しない情報伝達を実装

### テーマ対応

このカラーシステムは Material-UI の CSS Variables に対応しており、動的なテーマ切り替えをサポートしています。
